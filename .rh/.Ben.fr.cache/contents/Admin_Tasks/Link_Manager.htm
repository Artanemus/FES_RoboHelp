Linking front to back. How to link the FRONTEND FES_v###.accdb file to the BACKEND MSSQL EXPRESS database. Make sure you have completed the steps to building your ODBC connection, else this isn't going to work! SHIFT double CLICK Open the FRONTEND FES_v###.accdb file by ... Locate and highlight the file in MS Explorer. How down the SHIFT key. Keeping the SHIFT key pressed, double click the filename. Keep the SHIFT key press and wait for the Access application to fully load the file. When access is finished release SHIFT.    You can now see extended menus and all access objects.     If you see this warning message. Click 'Enable Content', close Access and repeat the above steps again.   If the warning keeps reappearing you might consider adjusting the 'Trust Centre Options' found in Access Options menu. NEXT   Select the 'External Data' tabsheet. Select 'Linked Table Manager'.     Click 'Select All'.  Check 'Always prompt for new location'. Press OK     Select the 'File Data Source' Location your FES.dsn file that you created in the Windows ODBC system utility. Press OK.     Success. Your ready to use FES
